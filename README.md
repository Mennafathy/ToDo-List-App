# ToDo-List-App
it's An application to check what do you plan to do in your day and then adds it to
your todos and mark completed task as it's done and it's in the list of completed tasks and
it make it easier to edit this task and then it's updated and saved

#prerequisite
it needs to install tailwindcss tool
store tasks in localstorage to prevent it from being lost
import icons style for delete and edit

#getting started
make loginform using tailwind
redirection to another page 
make todo app that can add,delete,edit,update and remove all tasks
checks if this task is checked put it int completed tasks list
if it's still in your todo put it in your pending list
